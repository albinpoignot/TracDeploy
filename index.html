TracDeploy by albinpoignot

Automatic Trac Deployment

TracDeploy is about give an easy to deploy Trac without a graphical user interface, on Linux systems. It's created for Gentoo based systems, and actually being tested on a Sabayon (v6). However, it should be cross-platform if you don't use the "Init packages" par which is the only one part dependent of the system. In this time, you require this tools (all other can be installed by the "Init packages part" : - perl To launch the tools, use this command : ./MANAGE_TRAC.sh [ []] If any question, or something else like critics, tell me :)

Dependencies

Python Perl Bash Subversion Trac Genshi Babel Apache

Install

No installation is needed. Copy the scripts in a folder and start using them. Please use MANAGE_TRAC.sh to call the other ones.
